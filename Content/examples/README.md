# Python Examples for VibeUE

This folder contains working Python code examples for common Unreal Engine operations.

## Organization

- **blueprint/** - Blueprint creation, variables, functions, nodes
- **asset/** - Asset management, search, create, delete
- **material/** - Material creation and node manipulation
- **level/** - Level actor operations
- **common/** - Shared patterns and utilities

## Usage

Examples are searched by the `get_python_examples` tool using:
- Filename keywords
- Category (folder name)
- Content search in file
- Tags in file comments

## Example Format

Each example should include:
```python
# Title: Brief description
# Category: folder name
# Tags: keyword1, keyword2, keyword3
# Description: More detailed explanation

import unreal

# Working code example here
```
